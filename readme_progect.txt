Правила размещения файлов проекта:

1. Корень папки src: 
    - страницы приложения, которые выводятся для показа пользователю, (наименование этих файлов = machines-supplies-page.php)

2. Папка PHP:
    - содержит файлы классов (папка class именуются: class_BasketProduct.php )
    - скрипты серверной обработки, без показа результатов пользователю (корнь папки и именуются: script_namemetod.php)
    - модульные секции страниц, используемые в нескольких страницах именуются /modules/(machines_supplies_section.php )

3. Папка JS:
    - react_class = содержит файлы классов формирования контента страниц (React) (папка class именуются: basketProduct.js )
    - modules = скрипты браузерной обработки, без показа результатов пользователю (корнь папки и именуются: script_namemetod.js), данные скрипты могут подключаться отдельно к страницам, на которой необходимо реализовать их потенциал
